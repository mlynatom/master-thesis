Následující kód spočítá průměr zadaných čísel. Kód používá funkce `sum()` a `len()` pro spočítání průměru.

```python
def spocti_prumer(cisla):
    """
    Spočítá průměr zadaných čísel.

    Args:
        cisla (list): Seznam čísel.

    Returns:
        float: Průměr zadaných čísel.
    """
    if not cisla:

    return sum(cisla) / len(cisla)
```
